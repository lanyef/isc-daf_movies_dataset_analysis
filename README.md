In this exercise, I did data preprocessing for a movies dataset. The goal? I Handled null and wrong  values, removing duplicate values and correcting inconsistencies and  errors in the dataset.



1. A movies table was created in already existing database and the movies.csv file was imported.
2. Duplicate values were removed using a windows function in a CTE.
3. For all the fields, leading and trailing white spaces where removed.
4. The field names of some fields were changed and also their  data types.
5. The year field which was changed to year\_released was standardized so that each row had the 'YYYY' date format.
